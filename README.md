# 狗刀电视

一个基于Godot引擎+Godot WRY插件的Windows版Y视频网页版播放器。

可通过游戏手柄实现上下键翻页换台，你懂的。

# 配置要求

## 最低配置要求

> CPU: i5-4590
> 
> GPU：Intel HD 4600
> 
> RAM：4GB 或更高
> 
> OS：Windows 10 22H2（需安装Edge WebView2框架）
> 
> 其他：兼容Xinput或Direct Input的手柄

## 推荐配置要求

> CPU：i7-7600
> 
> GPU：UHD 630
> 
> RAM：8GB 或更高
> 
> OS：Windows 11 24H2
> 
> 其他：兼容Xinput或Direct Input的手柄

### 其他配置要求

本软件支持RTX视频超分辨率功能，推荐使用NVIDIA RTX显卡并开启RTX视频超分辨率+RTX HDR，可大幅提升画面观感。

## 安装

如果你运行的操作系统为Windows 10，请先确保Microsoft Edge WebView2已经正确安装，否则软件将无法正常工作。

如已经安装，直接解压后运行即可。

## 已知问题

1. 使用墨将手柄作为换台遥控器的时候，在手柄自动关机时会导致软件卡死。经测试只有墨将手柄会导致此问题，因此怀疑是手柄固件设计缺陷。

   如使用墨将手柄，建议更换手柄。此问题无法定位和解决。

2. 本软件只是一个基于Godot Wry的浏览器。由于各方面的原因，无法保证此软件可以一直使用。受y视频网页版的版权等原因的限制，部分时段的节目可能会不能收看，这不是软件的问题。
3. 本软件没有提供广告拦截功能，如将来y视频在网页内嵌广告，则本软件也将显示广告。此广告不来源于本软件。
4. 根据相关法律法规要求，本软件**永不提供Android及任何TV版本**。如果你需要，请自己编译。
